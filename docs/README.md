## How to Access Documentation

### From Home Assistant Interface
1. Go to **Settings > Devices & Services**
2. Find your Eedomus integration
3. Click on the integration name
4. Look for the "Documentation" link (if available)

### From GitHub
- English: [configuration_documentation.md](configuration_documentation.md)
- French: [configuration_documentation_fr.md](configuration_documentation_fr.md)

### Language Selection
Home Assistant will automatically display the documentation in your interface language when possible. If you need to switch languages:

1. Go to **Settings > System > Language**
2. Change your Home Assistant language
3. Restart Home Assistant
4. The documentation links will point to the appropriate language version

## Troubleshooting

If you cannot find the documentation:
- Check your internet connection
- Verify that GitHub is accessible
- Try accessing the documentation directly from the GitHub links above
- Make sure you are logged in to GitHub if the repository is private

All documentation is kept up-to-date with the latest version of the integration. The documentation version matches the integration version specified in `manifest.json`.
